2. **Teeth whitening** - Before and after
3. **Dental implants** - Procedure or results
4. **Braces/Orthodontics** - Patient with braces

## Recommended Sources:
- Unsplash.com - Search "dental treatment", "teeth cleaning"
- Pexels.com - Search "dentist working"
- Pixabay.com - Search "dental care"

## Image Requirements:
- Format: JPEG
- Size: 1000px wide minimum
- File size: Under 180KB after optimization
- Aspect ratio: 4:3 or 16:9
