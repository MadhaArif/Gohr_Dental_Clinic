### Step 1: Search for Dental Clinic Interior Images
1. Go to [Google Images](https://images.google.com)
2. Type "modern dental clinic interior" in the search box
3. Press Enter

### Step 2: Apply Usage Rights Filter
1. Click "Tools" below the search bar
2. Click "Usage Rights"
3. Select "Creative Commons licenses"
4. You'll now see only images that are free to use

### Step 3: Select and Verify an Image
1. Click on any image that looks professional
2. Check the image details page for licensing information
3. Look for phrases like:
   - "Labeled for reuse"
   - "Labeled for noncommercial reuse"
   - "Creative Commons"

### Step 4: Download the Image
1. Right-click on the image
2. Select "Save image as..." or "Save picture as..."
3. Save to your computer

### Step 5: Optimize the Image
1. Go to [TinyJPG.com](https://tinyjpg.com)
2. Drag your downloaded image to the website
3. Wait for optimization
4. Download the compressed version

### Step 6: Place in Correct Folder
Move your optimized image to:
```
/public/images/clinic/clinic-interior-1.jpg
```
